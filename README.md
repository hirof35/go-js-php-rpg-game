# ⚔️ リアルタイムコマンド選択型RPGミニゲーム

Go（バックエンド/ステート管理）と PHP（ロジック計算）、HTML/JS（フロントエンド）を組み合わせた、リアルタイム同期型のバトルミニゲームです。

## 🌟 特徴
- **3層アーキテクチャ**: Go言語のメモリ上でゲームの状態（HP/MP等）を完全管理。
- **リアルタイム同期**: SSE (Server-Sent Events) を採用し、画面をリロードせずに敵やプレイヤーの状態が即座に反映。
- **戦略的な3つの回復手段**: MPを消費する「小回復」「大回復」と、回数制限のある「薬草」を駆使してボスを倒す駆け引き。
- **リトライ機能**: ゲームクリア・ゲームオーバー画面からシームレスに初戦から再戦可能。

## 🛠️ 開発環境 / 技術スタック
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS / EventSource)
- **Backend (State)**: Go 1.2x (net/http, sync.Mutex)
- **Backend (Logic)**: PHP 8.x (cURL)
